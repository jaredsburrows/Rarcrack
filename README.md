## Compile

    make

## Usage

	$ ./rarcrack 
	RarCrack! 0.2 by David Zoltan Kedves (kedazo@gmail.com)

	USAGE: rarcrack encrypted_archive.ext [--threads NUM] [--type rar|zip|7z]
	       For more information please run "rarcrack --help"
